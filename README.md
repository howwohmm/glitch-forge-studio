# glitch-forge-studio

> Creative image and video effect editor with GPU-accelerated glitch presets.

![Status: Experiment](https://img.shields.io/badge/Status-Experiment-yellow)

## What It Does

Upload images or videos and apply real-time visual effects. Features 15+ GPU-accelerated presets across categories like Error Diffusion, Bitmap, Glitch, Pattern, and Modulation. Share and discover community presets.

## Tech Stack

`React` `Vite` `TypeScript` `Supabase` `shadcn/ui` `Tailwind CSS` `Framer Motion` `React Dropzone` `TanStack Query`
